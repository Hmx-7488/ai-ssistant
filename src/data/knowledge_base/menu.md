# 饭小二餐厅菜单

## 招牌菜

### 🔥 招牌红烧肉
- **价格**：58元/份
- **描述**：精选五花肉，慢炖3小时，入口即化，肥而不腻。搭配秘制酱汁，甜咸适中，是米饭的最佳伴侣。
- **辣度**：不辣
- **分量**：约400g，适合2-3人分享

### 🐟 秘制烤鱼
- **价格**：88元/份
- **描述**：新鲜鲈鱼，炭火慢烤，外焦里嫩。搭配独家秘制酱料，鲜香四溢。配菜有豆芽、藕片、木耳等。
- **辣度**：可选微辣/中辣/特辣
- **分量**：整条约800g，适合2-4人分享

### 🥩 水煮牛肉
- **价格**：62元/份
- **描述**：选用上等牛里脊，搭配花椒、干辣椒，麻辣鲜香。肉质嫩滑，汤汁浓郁。
- **辣度**：中辣（可调整）
- **分量**：约350g，适合2-3人分享
